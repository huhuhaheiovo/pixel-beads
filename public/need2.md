# Pixel Beads (pixelbeads.pics) 网站分析报告

Pixel Beads 是一个免费的基于 Web 的工具，旨在帮助拼豆爱好者和艺术家将照片或图像转换为物理“拼豆”模式（支持 Perler, Hama, 和 Artkal 等品牌）。

## 核心功能分析

1. **图像转换与处理 (Image Conversion & Processing)**
   - **标准模式**：支持上传 JPG/PNG 图像（最大 10MB）并立即生成像素图。
   - **裁剪/草稿模式 (`/crop-page`)**：提供专门的工具，允许用户在生成模式前对图像进行裁剪、缩放，并与网格精确对齐。
   - **实时预览**：在用户调整设置时，系统会即时更新转换结果。

2. **模式高度自定义 (Pattern Customization)**
   - **品牌调色板**：预设了 **Perler, Hama, 和 Artkal** 等主流国际品牌的拼豆颜色库。
   - **网格控制**：网格尺寸可调，范围从 **10×10 到 100×100**。
   - **颜色灵敏度调整**：可微调算法将图像颜色匹配到拼豆颜色的精确度。

3. **手动编辑工具 (Manual Editing Tools)**
   - **画笔与填充工具**：与纯自动转换器不同，该网站提供画笔和填充功能，允许用户手动修正或修改网格上的特定拼豆颜色。
   - **撤销/重做 (Undo/Redo)**：支持撤销操作，方便用户进行精细化调整。

4. **项目管理与产出 (Project Management & Output)**
   - **自动拼豆计数器**：列出项目所需的所有颜色及其精确的拼豆数量，方便准备材料。
   - **可下载的模式图**：生成高分辨率、可打印的网格图纸，包含每颗拼豆对应的颜色代码。

## 网站特色与对比

与同类工具（如 `pixelbeads.org`）相比，`pixelbeads.pics` 的特点如下：

| 维度 | pixelbeads.pics | pixelbeads.org |
| :--- | :--- | :--- |
| **主要定位** | 强调易用性和交互性 | 强调技术参数和算法控制 |
| **手动编辑** | **是**（提供画笔/填充工具） | 否（主要依靠滑块参数） |
| **品牌支持** | 侧重国际品牌 (Perler, Hama, Artkal) | 侧重更多样化品牌 (MARD, COCO, ManMan 等) |
| **独特功能** | 提供专门的裁剪/对齐页面 | 支持 CSV 数据文件导出 |
| **交互体验** | 现代、流线型、简洁 | 界面更为传统，提供“高级设置” |

## 总结

`pixelbeads.pics` 是一个更注重**用户体验和互动编辑**的工具。其核心优势在于强大的**手动编辑功能**，使用户能够在不使用外部像素画软件的情况下，直接在网页上对生成的图纸进行最后修饰，非常适合追求细节的拼豆玩家。
