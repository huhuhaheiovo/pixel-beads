请按高级 SEO 标准优化并生成内容，需满足以下全部要求：

【网站地址】
- https://www.pixel-beads.com

【搜索意图（Three Cs）】
- Content 必须严格符合搜索意图的 three Cs：
  - Content type（内容类型）与当前 SERP 主流一致
  - Content format（格式）与用户期望一致（如列表 / 指南 / 工具说明）
  - Content angle（角度）与排名靠前页面一致，但提供差异化价值
- 明确覆盖搜索者最关心、最想快速得到的信息
- 相比当前排名靠前页面，提供至少 1–2 个独特视角或补充信息
- 内容需体现专业性或第一手经验（如真实使用场景、实践建议）


URL 需要使用描述性 slug，与页面主关键词一致，简短可读，
避免参数、编号或无意义字符，利于用户理解和 SEO。

【关键词策略】
- 主关键词：路由地址为主关键词，例如/aqua-beads;/fuse-beads;/image-to-pixel
- 一个页面一个关键词
- 禁止关键词堆砌


【内容结构（On-page SEO）】
- 使用 H1 作为页面主标题（仅 1 个）
- 使用 H2–H6 作为分级小标题，结构清晰
- 内容逻辑清楚，便于快速扫描（适合 Featured Snippet）
- 语言自然，避免 AI 痕迹和营销腔

【Meta 标签】
- Title：
  - 包含主关键词
  - 有吸引力，符合搜索意图
  - ≤ 60 字符
- Meta Description：
  - 与搜索意图强一致
  - 提高 CTR
  - ≤ 160 字符
  - 非关键词堆砌



【链接策略】
- 包含相关的站内链接（帮助用户继续操作或理解）
- 包含权威或相关的外部引用链接（如工具、资料来源）
- 外链需服务于内容目的，而非纯 SEO

【图片优化】
- 图片文件名具备描述性（而非 img_001）
- 所有图片包含准确、自然的 alt 文本
- 图片体积需可压缩、适合网页加载
- 图片内容需真实服务于页面理解

【Advanced SEO 要求】
- 内容结构适合 Featured Snippets（列表、定义、步骤清晰）
- 推荐合适的 Schema Markup（如 FAQ / HowTo / SoftwareApplication）
- 页面设计需符合 Core Web Vitals 优化原则
- 默认使用 HTTPS
- 页面需移动端友好（Mobile-first）
- 不使用影响体验的弹窗或插屏（法律要求除外）

【输出语言与风格】
- 语言：{英文 / 中文}
- 风格：专业但自然，像真实从业者或用户
- 避免夸张营销词（如 ultimate / best ever）
