# 拼豆图纸生成器 UI 优化更新

我们很高兴地宣布，拼豆图纸生成器完成了一次重要的功能更新和 UI 优化！本次更新主要聚焦于提升创作体验和视觉效果，让您能够更灵活地预览和导出拼豆图纸。

## 新增拼豆样式选择功能

### 三种样式可选

现在您可以在生成器中自由选择三种不同的拼豆样式：

#### 1. 网格方珠（Square）

经典的方形拼豆样式，这是最传统的拼豆外观。适合：
- 传统拼豆作品
- 需要清晰网格线的设计
- 追求经典像素艺术风格

#### 2. 网格圆珠（Round）

圆形的拼豆样式，视觉效果更加柔和。适合：
- 现代风格的拼豆作品
- 需要更自然外观的设计
- 追求柔和视觉效果的项目

#### 3. 网格中空珠（Hollow）

中空的拼豆样式，独特的视觉效果。适合：
- 创意拼豆作品
- 需要特殊视觉效果的设计
- 追求独特风格的项目

### 实时预览

选择不同的拼豆样式后，您可以在画布上实时看到效果，无需等待导出就能预览最终效果。这让您能够：

- **快速对比**：轻松切换不同样式，找到最适合的视觉效果
- **即时调整**：根据预览效果调整设计，提高创作效率
- **精准预览**：所见即所得，确保最终效果符合预期

## 如何使用新功能

### 选择拼豆样式

1. 打开[拼豆图纸生成器](/zh/perler-bead-pattern-generator)
2. 在右侧设置面板中找到「拼豆样式」选项
3. 点击选择您想要的样式（方形、圆形或中空）
4. 画布上的预览会立即更新

### 导出不同样式

导出时，您选择的样式会应用到导出的 PDF 或图片中：

- **PDF 导出**：生成的 PDF 文件会使用您选择的样式
- **图片导出**：导出的图片会反映您选择的样式效果
